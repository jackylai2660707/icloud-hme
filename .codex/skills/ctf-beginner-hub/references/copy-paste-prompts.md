# 可直接复制使用的提示词

## 模式 1：自动分流

```text
请用 ctf-beginner-hub 的自动模式帮我处理这道题。
要求：
1. 先判断它最像哪一类 CTF 题
2. 给出主候选 skill 和次候选 skill
3. 解释为什么这么判断
4. 只给我现在最该做的前 3 步
5. 每一步都用小白能看懂的话解释

题目信息：
[把题面、附件名、URL、IP:PORT、源码结构粘贴在这里]
```

## 模式 2：先头脑风暴，再决定

```text
请用 ctf-beginner-hub 的头脑风暴模式带我看这道题。
我现在不确定它属于什么类型，也不知道第一步做什么。
请先帮我梳理：
1. 题目到底想让我做什么
2. 我现在最缺的关键信息是什么
3. 应该自动选 skill 还是手动选 skill
4. 如果要手动选，最可能是哪两个 skill

题目信息：
[把题面、截图、描述、自己做过的尝试粘贴在这里]
```

## 模式 3：手动指定

```text
我判断这题更像 [ctf-web / ctf-reverse / ctf-crypto / ctf-pwn / ctf-forensics / ctf-misc / ctf-osint / ctf-malware / ctf-ai-ml]。
请直接按这个方向继续，但如果你觉得我判断错了，请先告诉我为什么。

题目信息：
[粘贴内容]
```
