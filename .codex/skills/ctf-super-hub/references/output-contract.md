# Output Contract

Use this default structure for beginner-friendly answers.

## 1. Current read
One sentence on what the challenge most likely is.

## 2. Why this route
Give 2-4 concrete signals.

## 3. Next 1-3 steps
Keep the list short.

## 4. What each step is doing
Explain each step in plain language.

## 5. If it fails
Give the next branch instead of saying only "try again".

## 6. Tiny glossary
Define unfamiliar terms in one line each.

## Confidence block
When confidence is not high, include:
- Primary skill
- Backup skill
- Missing evidence
- Smallest discriminating action
